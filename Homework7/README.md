Цей файл існує для того,щоб закріпити різницю між Number, parseInt, parseFloat та показати, як вони поводяться з "брудними" рядками.Також вона потрібна,  щоб побачити свій вік через 5 років.
У деяких випадках parseInt() та parseFloat() виводить NaN, коли не може знайти числове значення.Наприклад: abc123, у цьому виразі літери йдуть першими, і через це він не може знайти числове значення.Так само і з UAH 999
Щоб запустити цю програму, треба відкрити index.html.
У цій програмі є декілька функцій, а саме: prompt, Number, console.log(), trim(), Number.isNaN(), alert, parseInt, parseFloat .
